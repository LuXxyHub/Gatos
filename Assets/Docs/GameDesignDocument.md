# Cosmic Yarn Cat - Game Design Document

## 1. Game Overview
**Title:** Cosmic Yarn Cat
**Genre:** Isometric Action RPG
**Platform:** PC (Unity)
**Target Audience:** Fans of cozy exploration games with action elements (e.g., Tunic, Cat Quest).

## 2. Core Concept
A space-traveling cat explores a universe of Yarn Moons. The goal is to defeat textile enemies, collect yarn, and assemble the Giant Lunar Yarn Cat to grant 9 wishes.

## 3. Visual Aesthetic
- **Style:** Handcrafted, textile, woven. Everything looks like it's made of fabric, wool, felt, or thread.
- **Lighting:** Soft space lighting, pastel colors, stardust glow.
- **Effects:** Thread particles, unraveling enemies, stitching animations.

## 4. Gameplay Mechanics
### 4.1 Exploration
- **Isometric View:** Fixed camera angle.
- **Movement:** Analog movement.
- **Abilities:**
    - **Boosted Jump:** Use yarn elasticity to jump higher/further.
    - **Cosmic Scratch:** Basic melee attack.
    - **Charged Claw:** Heavy attack breaking shields.
    - **Quick Reflex:** Dodge/Dash with invulnerability frames.

### 4.2 Combat
- Real-time action.
- Enemies drop "Stellar Thread" (XP/Currency).
- **Yarn-Based Armor:** Craftable suits with stats (e.g., "Woolen Knight", "Silk Assassin").

### 4.3 Progression
- **Skill Tree:** Unlock new moves (e.g., "Needle Throw", "Yarn Ball Roll").
- **Moon Collection:** Defeat the boss of a moon to claim it.

## 5. Enemies
- **Yarn UFO:** Floating balls shooting thread lasers.
- **Needle Sentinel:** Robotic needles that dash/spear.
- **Space Moth:** Corrodes fabric armor.
- **Weave Worm:** Burrows underground.
- **Living Knot:** Traps player.
- **Ancient Weaver (Spider):** Boss type.
- **Felt Mammoth:** Tank enemy.

## 6. World Design
- **Biomes:**
    - *The Silken Fields* (Starter)
    - *The Knotted Canyons*
    - *The Velvet Void*
- **Interactive Elements:**
    - **Elastic Platforms:** Bounce pads.
    - **Knot Traps:** Hazards.
    - **Woven Bridges:** Unfoldable paths.

## 7. Technical Pillars
- **Unity Engine**
- **New Input System**
- **ScriptableObject Architecture** for Items/Stats.
